# Telegram Shop Bot (Database Course Project)

Проєкт створено студентом Крупка Іван (ІПЗ 3/2) як частина курсової роботи з баз даних.

##  Функціонал
- Перегляд каталогу товарів
- Кошик і оформлення замовлення
- PDF-квитанції
- Оплата та доставка
- Панель адміністратора

##  Структура Бази Даних
(ERD.png або таблиці з розділу 3)

##  Запуск проєкту
1. Клонувати репозиторій
2. Встановити залежності
3. Ініціалізувати базу даних
4. Додати TOKEN у файл `.env`
5. Запустити `bot.py`

##  Technologies
- Python
- SQLite/PostgreSQL
- Telegram Bot API
- ReportLab (PDF)
